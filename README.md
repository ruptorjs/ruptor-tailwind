# Ruptor Tailwind

Tailwind extension for [Ruptor](https://ruptor.net).

## Installation

1. Inside your Ruptor project, run `npm install -D @ruptor/tailwind`
2. Add **@ruptor/tailwind** to the extension field inside your Ruptor config in package.json.

```json
{
  "ruptor": {
    "extensions": [
      "@ruptor/tailwind"
    ]
  }
}
```
